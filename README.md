# vba_code_prettier_test

## 使用する npm パッケージ

### ESLint

#### インストール

```
yarn add -D eslint
```

#### 初期設定

```
yarn run eslint --init
```

#### prettier 連携

```
yarn add -D eslint-config-prettier
```

### prettier

#### 用途

コード整形

#### インストール

```
yarn add -D prettier
```
